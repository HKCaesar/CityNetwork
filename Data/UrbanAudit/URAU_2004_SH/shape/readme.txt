==========================
  GISCO Urban Audit 2004
==========================


Data
====

The GISCO Urban Audit 2004 data is disseminated in ESRI Geodatabase format (Access database).


Metadata
========

The metadata follows the ISO 19139 recommendations. It is provided in XML format.

To visualise the URAU metadata just open the file "URAU-MetadataCatalog.htm" in your web browser.

Due to the restrictions in the length of the attribute names in shapefiles (max. 10 characters), some attribute names in the metadata might slightly differ from the actual attribute names in the shapefiles.


Data source and copyright acknowledgement.
==========================================

GISCO Urban Audit 2004 is a geographical dataset developed by the European Commission based on EuroBoundary Map 2004 and generalised to 1:3 000 000 scale.

When the GISCO Urban Audit 2004 geographical dataset is used in any printed or electronic publication, the data source should be acknowledged in the legend of the map and in the introductory page of the publication.

The data source notice on the legend of the map and the introductory part of the publication is as follows:

	Data source: GISCO - Eurostat (European Commission)




Copyright clauses for geographical data derived from data developed at EuroGeographics (EG).
============================================================================================

The GISCO Urban Audit 2004 geographical dataset contains geometry derived from EuroBoundary Map 2004 (EBM2004) developed by EuroGeographics. 

Where polygon geometry from the GISCO Urban Audit 2004 geographical dataset is used in any printed or electronic publication, the copyright on the source data set should be acknowledged in the legend of the map and in the introductory page of the publication, following the European Commission data source acknowledgement.

The copyright notice on the legend of the map and the introductory part of the publication is as follows:

© EuroGeographics for the administrative boundaries

For publications in languages other than English, the translation of the appropriate source and copyright notices in the language of the publication shall be used.
